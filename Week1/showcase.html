PyTorch & Tensors

🔥

What is PyTorch?

Open-source ML framework by Meta. Industry standard used by Tesla, Meta, OpenAI.
💡 Analogy: LEGO blocks for AI — each block connects to build complex systems
Python Native · Dynamic Graphs · GPU Support · Large Community

What is a Tensor?

Multi-dimensional array of numbers — the core data structure
0D Scalar → Single number (5)
1D Vector → List [1, 2, 3]
2D Matrix → Table [[1,2], [3,4]]
3D Cube → Image (Height × Width × Channels)
4D Batch → Batch of images

Autograd & Neural Networks

Autograd

Automatic differentiation engine — computes gradients automatically
💡 Analogy: Driving down a hill blindfolded — tells you which direction is DOWN
1. Forward Pass → Model predicts
2. Calculate Loss → See how wrong
3. Backward Pass → Autograd computes gradients
4. Update Weights → Using gradients

nn.Module — Build Neural Networks

Base class for all neural network modules
Two Required Methods:
• __init__() — Define your layers
• forward() — Define how data flows

Loss Functions & Optimizers

Loss Functions

Measures how wrong the model is → Lower is better
Task Loss Function
Multi-class Classification CrossEntropyLoss
Binary Classification BCEWithLogitsLoss
Regression MSELoss
💡 Analogy: Loss is like a video game score — you want the LOWEST score possible

Optimizers

Updates model weights to reduce loss
Optimizer Best For
Adam Most problems — fast convergence
SGD Simple, well-understood
AdamW Better regularization
Training Step:
Zero Grad → Forward → Loss → Backward → Update
