PyTorch & Tensors

What is PyTorch?

Open-source machine learning framework developed by Meta AI Research. Industry standard used by Tesla, Meta, OpenAI, and other leading organizations.
Analogy: Structural framework for AI development — each component connects to build complex systems
Python Native Dynamic Graphs GPU Support Large Community

What is a Tensor?

Multi-dimensional array of numbers — the fundamental data structure in PyTorch
0D Scalar → Single number
1D Vector → List of numbers
2D Matrix → Table of numbers
3D Cube → Image (Height × Width × Channels)
4D Batch → Multiple images

Autograd & Neural Networks

Autograd

Automatic differentiation engine that computes gradients of tensor operations
Analogy: Navigation system for optimization — provides directional guidance to minimize loss
1. Forward Pass → Model produces predictions
2. Calculate Loss → Measure prediction error
3. Backward Pass → Autograd computes gradients
4. Update Weights → Optimize model parameters

nn.Module

Base class for all neural network modules in PyTorch
Two Required Methods:
— __init__() Define neural network layers
— forward() Define data flow through the network
Linear
Conv2d
MaxPool2d
BatchNorm2d

Loss Functions & Optimizers

Loss Functions

Measure prediction error — lower values indicate better performance
Task Loss Function
Multi-class Classification CrossEntropyLoss
Binary Classification BCEWithLogitsLoss
Regression MSELoss
Analogy: Performance score — the objective is to minimize this score through training

Optimizers

Update model weights to reduce loss over time
Optimizer Application
Adam General purpose — fast convergence
SGD Simple, well-understood approach
AdamW Improved regularization
Training Step Sequence:
Zero Grad → Forward Pass → Loss Calculation → Backward Pass → Weight Update
